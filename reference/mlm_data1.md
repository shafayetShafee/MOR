# A Simulated Multilevel Data

This data contains data of 100 clusters where each cluster contains 50
individuals.

## Usage

``` r
mlm_data1
```

## Format

A tibble with 5000 rows and 5 variables:

- `cluster`:

  Cluster ID

- `id`:

  Individual ID

- `X1c`:

  A continuous covariate

- `X2b`:

  A binary covariate

- `Yij`:

  Response Variable
